    alert 명령어 설명

    1. alert 는 dialog 창을 열고 메시지를 표시한다.
    2. 단순히 메시지를 전달하는 기능을 수행한다.</script>태그에 작성
    3. 사용자의 입력은 확인할 수 없다.

    if~else 명령어 설명

    if - 조건에 맞는 명령문을 실행
    if ~ else - 한 개 이상의 조건을 설정하여 그에 맞는 명령문 실행

    if~else 연습

    if (!value) { result.innerHTML = '값을 적어주세요.' } else if (value <= 10) { result.innerHTML = '값이 10이상입니다.' } else if (value <= 100) { result.innerHTML = '값이 10이상이고 100이하입니다.' } else { result.innerHTML = '값이 100보다 큽니다.' }

    switch~case

    값에 따라 서로다른 코드를 실행할 때, switch 문이 적합하다

    switch~case 연습

    switch (value) { case '사과': result.innerHTML = 200 break case '포도': result.innerHTML = 300 break case '체리': result.innerHTML = 400 break default: result.innerHTML = '사과, 포도, 체리 중 하나를 적어주세요' }

    for

  • for 명령은 반복횟수가 알거나 주어진 경우 사용하는 반복문이다.
  • for(초기값 설정; 조건식; 반복 후 작업) { 작업문 }
    • if (value > 10) { result.innerHTML = '10 이하의 수를 적으세요.' } else { for (let i = 0; i < Number(value); i++) { result.innerHTML += i + ' ' } }
